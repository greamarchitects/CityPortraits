# Datasets

This section documents the primary data sources used for urban analysis and modeling in the CityPortraits project.

## Dataset Categories

### Spatial Data
- **[GHSL](ghsl.md)** - Global Human Settlement Layer: Built-up area mapping and population density
- **[OSM](osm.md)** - OpenStreetMap: Street networks, points of interest, and land use classification

### Administrative Data
- **[ASIG](asig.md)** - Albanian Spatial Information Gateway: Official spatial data and administrative boundaries
- **[GDE](gde.md)** - Geographic Data Environment: Albanian national geospatial data platform

### Socio-Economic Data
- **[Census](census.md)** - Population and housing statistics from national census data
